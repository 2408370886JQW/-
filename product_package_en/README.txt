Product Prototype Package - File Mapping Guide
产品原型文件包 - 文件对照指南

This package contains screenshots of the product prototype, organized by functional flows.
To ensure compatibility across all operating systems, folder names are in English.
Please refer to the mapping below for Chinese descriptions.
